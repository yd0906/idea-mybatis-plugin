下载后是zip解压后目录为mybatis_plus-master，点进去mybatis_plus.zip

解压这个zip文件，找到mybatis_plus

安装mybatis plugin插件

    有两种方法可以去安装。

	    第一种适合windows（不要打开IDEA ）

		    第①步，打开我的电脑，找到路径C:\Users\Administrator\.IntelliJIdea2018.1\config\plugins，这是本人的电脑路径，你的应该是C:\{用户}\{USER}\.IntelliJIdea{VERSION}\config\plugins

		    第②步，把上面下载后的mybatis_plus文件夹（也就是点击mybatis_plus-master目录里解压mybatis_plus.zip后的文件夹），是文件夹哦，不要带上ZIP， 复制到第一步的路径下

		    第③步，打开我的电脑，找到路径C:\Users\Administrator\.IntelliJIdea2018.1\options，这是本人的电脑路径，
				  你的应该是C:\{用户}\{USER}\.IntelliJIdea{VERSION}\config\options，把刚下载的文件夹中的mybatis.xml复制到该文件夹下

 
    第二种方法适合windows或mac平台（速度最快，最方便）

		    第①步，打开IDEA，点击File -- Settings -- plugins -- install plugin from disk

		    第②步，找到刚才mybatis_plus.zip文件 选择它
